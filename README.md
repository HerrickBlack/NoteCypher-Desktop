# NoteCypher Desktop  
  
A native cross-platform PDF note optimizer built with Rust and Iced.  
  
## Features  
  
- **100%% Offline** - No internet connection required  
- **Cross-Platform** - Windows, macOS, and Linux  
- **Native Performance** - Fast PDF processing  
- **Multiple PDF Upload** - Load multiple PDF files  
- **Interactive Selection** - Select pages with thumbnails  
- **Image Filters** - Invert, Clear BG, Grayscale  
- **Flexible Layout** - 1-6 slides per page  
- **Dark Mode** - Light/dark themes  
  
## Quick Start  
  
```bash  
cargo run  
```  
  
Or build release:  
  
```bash  
cargo build --release  
```  
  
See [QUICKSTART_FULL.md](QUICKSTART_FULL.md) for detailed instructions.
